CSS

I decided to create two CSS transitions for this assignment.

Transition 1

Here is the code snippet I used to create my first CSS transition:

nav {
   display: flex;
   gap: 0.5rem;
}

.nav-container {
   flex: 1;
   background-color: #333;
   color: #fff;
   border: 1px solid;
   padding: 0.5rem;
   text-align: center;
   text-decoration: none;
   transition: all 0.5s ease-out;
}

.nav-container:hover,
.nav-container:focus {
   background-color: #FBFBFF;
   color: #333;
}
